# 📖 ThePower_Katas_Python: Descripción 
En este proyecto apliqué los conocimientos adquiridos en el módulo Python y en las clases online, resolviendo ejercicios prácticos (katas) para reforzar conceptos de programación.
# 💻 Requerimientos/Conocimientos aplicados
Este proyecto demuestra el manejo de los siguientes conceptos:
1. Tipos de datos básicos y funciones incorporadas.
2. Estructuras de datos en Python y sus métodos.
3. Condicionales.
4. Estructuras de iteración.
5. Funciones definidas en Python.
6. Clases y entendimiento de la programación orientada a objetos.
7. Uso de módulos y librerías estándar (functools, collections, re).
8. Aplicación de buenas prácticas de programación.
